# Backend - Full-Stack Assessment

This is the backend API for the Full-Stack Assessment Assignment.  
It is built using **Laravel 10** and **MySQL**.

## Features

- Fetch all products
- Fetch single product details
- Create orders with customer info and multiple items
- Calculates total order amount
- Proper validation and error handling
- Clean RESTful API structure

## Tech Stack

- Laravel 10
- PHP 8+
- MySQL
- Composer
